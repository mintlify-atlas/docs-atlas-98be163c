DirectPackageInstaller (DPI) is a cross-platform desktop tool that lets you send PKG files directly to your PS4 or PS5 without manually copying files. You paste a URL — whether it's a plain direct link, a debrid-service link, a compressed archive, or a PSN update manifest — and DPI resolves it, optionally proxies the download through your PC over LAN, and pushes it to your console's package installer. It runs on Windows, Linux, macOS, and Android, and requires no configuration beyond knowing your console's IP address.

## Installation methods

DPI communicates with your PS4 or PS5 through one of three methods. You only need one of them active on your console when using DPI.

- **Remote Package Installer** — A PS4 homebrew app that exposes an HTTP API on port 12800. DPI sends PKG download tasks directly to this API.
- **GoldHEN Payload Server** — DPI delivers a payload to your console over ports 9090, 9021, or 9020. No separate homebrew app required beyond GoldHEN itself.
- **etaHEN DPIv2 API** — Targeted at PS5 users running etaHEN. DPI sends packages via multipart POST to port 12800 using the DPIv2 protocol.

## Supported platforms

| Platform | Architectures | .NET 8 Runtime required |
|----------|--------------|------------------------|
| Windows  | ARM, ARM64, x64, x86 | Yes |
| Linux    | ARM, ARM64, x64 | Yes |
| macOS    | ARM64, x64 | No (self-contained) |
| Android  | ARM, ARM64, x64, x86 | Yes |

<Note>
  DPI must run on a device that is on the **same local network** as your PS4 or PS5. The console communicates with DPI's local HTTP server, so network isolation (such as guest Wi-Fi) will prevent installation.
</Note>

## Supported file hosting services

DPI works with a broad range of sources out of the box.

- **Any direct link** — No authentication required. Paste and go.
- **AllDebrid** — Requires an [API key](https://alldebrid.com/apikeys/).
- **RealDebrid** — Requires an [API key](https://real-debrid.com/apitoken).
- **DebridLink** — Requires an [API key](https://debrid-link.com/webapp/apikey).
- **Google Drive** — Public and shared files supported.
- **Mediafire** — Direct download resolution handled automatically.
- **PixelDrain** — Supported without additional configuration.
- **1Fichier** — Supported, but enforces a 60-minute cooldown between downloads.

For the best download speeds, AllDebrid, RealDebrid, Google Drive, and Mediafire are recommended. 1Fichier is the slowest option.
